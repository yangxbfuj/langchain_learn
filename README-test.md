# DeepAgent

DeepAgent 是一个强大的 AI 代理，能够使用工具帮助用户完成各种任务。

## 核心特性

### 🧠 智能任务管理
- 自动规划复杂任务，将大目标拆解为可执行的小步骤
- 通过 Todo 列表实时追踪任务进度
- 支持并行执行独立任务，大幅提升效率

### 🔧 丰富的工具集成
- **文件系统操作**：读取、写入、编辑文件，支持 glob 模式搜索和 grep 文本搜索
- **子代理系统**：可启动短生命周期的子代理处理隔离任务，避免上下文污染
- **代码执行与搜索**：在代码库中高效导航和分析

### 🚀 高效工作流
- 简洁直接，不过度解释
- 遵循现有代码风格和项目约定
- 遇到问题自动分析和迭代，不重复无效尝试

### 🔍 专业客观
- 以准确性为优先，敢于礼貌地指出错误
- 避免不必要的赞美或情感化表达
- 在信息不足时主动提出最小必要的跟进问题

### 📁 项目感知
- 先阅读文件再编辑，确保理解现有内容
- 支持大型文件的分页读取
- 多模态文件支持（图片、音频、视频、PDF）

### ⚡ 并行处理
- 尽可能并行化工具调用和子任务
- 独立的步骤可同时进行，节省用户时间
- 高效的上下文管理，减少 token 消耗

## 开始使用

只需向 DeepAgent 描述你的需求，它会自动选择合适的工具和策略来完成任务。
